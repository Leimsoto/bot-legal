🔒 Política de Privacidad

Cat’s Bot no almacena información personal de los usuarios. Solo procesa temporalmente datos necesarios para su funcionamiento, como:

  • IDs de usuario, canal o servidor (solo para ejecutar comandos).
  • Mensajes procesados por funciones de moderación o chat IA (sin almacenamiento permanente).

Los datos nunca se comparten con terceros y se manejan únicamente mediante la API oficial de Discord.
